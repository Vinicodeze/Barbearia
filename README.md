## Black BarberShop

Este é um projeto de site para uma barbearia 

## Sobre o projeto
O objetivo é criar um site moderno e responsivo para uma barbearia, com foco em:
- Apresentação dos serviços
- Galeria de cortes
- Informações de contato
- Layout simples e profissional

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- Página inicial estilizada
- Carrossel de imagens
- Seção de serviços
- Galeria de imagens
- Footer com informações da barbearia

##  Objetivo
Este projeto faz parte do meu aprendizado em desenvolvimento web e serve como portfólio.
